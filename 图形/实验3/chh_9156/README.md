# 汉诺塔 Unity 工程 — chh_9156

## 使用方法（3 步搞定）

### ① 创建项目
打开 **Tuanjie Hub**（`C:\AI\Tuanjie Hub\Tuanjie Hub.exe`）
→ 新建项目 → 选择 **3D 模板**
→ 项目名称填 `chh_9156`
→ 点击创建

### ② 导入脚本
将本目录下的 `Assets` 文件夹**整个复制**到你刚创建的项目根目录
（覆盖同名 `Assets` 文件夹）

### ③ 一键搭建场景
回到 Tuanjie 编辑器 → 顶部菜单栏 → **Tools → Setup Hanoi Game**
→ 点"开始搭建" → 等 2 秒

✅ 完成！点击顶部 ▶ **Play** 按钮即可开始游戏。

---

## 游戏操作说明

| 操作 | 效果 |
|------|------|
| 点击柱子顶部的圆盘 | 圆盘变黄（选中状态） |
| 点击目标柱子 | 圆盘飞过去（抬升→平移→下降动画） |
| 再次点击同一个圆盘 | 取消选中 |
| 三张盘全移到 **C 柱(右)** | 显示"恭喜通关！" |
| 点击右上角 **重置按钮** | 恢复初始状态，步数归零 |

## 工程文件结构

```
Assets/
├── Scripts/
│   ├── chh_Disk.cs           ← 圆盘：点击检测 + 高亮 + 三段动画
│   ├── chh_Pillar.cs         ← 柱子：点击检测 + 栈结构管盘
│   ├── chh_GameManager.cs    ← 总控：状态机 + 合法性判定 + UI更新
│   └── Editor/
│       └── chh_SceneSetup.cs ← 一键搭建工具（Tools 菜单）
├── Materials/                 ← 自动生成的材质（棕/灰/蓝/绿/红）
├── Prefabs/                   ← 自动生成的圆盘预制体
└── Scenes/MainScene.unity     ← 自动生成/保存的场景文件
```
